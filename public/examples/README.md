## Formato de Imágenes

- **Formato recomendado**: JPG o WebP
- **Tamaño recomendado**: Ancho mínimo 800px, máximo 1920px
- **Optimización**: Comprimir las imágenes para web (calidad 80-85%)
- **Aspect ratio**: Las imágenes se mostrarán con `object-fit: contain` para evitar recortes

## Cómo Agregar las Imágenes

1. Guarda cada imagen con el nombre exacto especificado arriba
2. Coloca todas las imágenes en esta carpeta: `public/examples/`
3. Las imágenes se cargarán automáticamente en la galería

## Troubleshooting

Si una imagen no se muestra:
- Verifica que el nombre del archivo sea exactamente como se especifica (incluyendo la extensión)
- Asegúrate de que la imagen esté en formato JPG, PNG o WebP
- Revisa que la imagen no esté corrupta
- Verifica los permisos de lectura del archivo

## Estructura del Componente

Las imágenes se utilizan en el componente `ExamplesGallery.jsx` que se abre al hacer clic en el card "Anuncio" del Momento 4.
